# Vue IS

Infinite scroll with Vue

![](https://github.com/weiying-chen/vue-is/blob/main/screenshot.png)

**[Live Demo](https://weiying-chen.github.io/vue-is/)**

## Installation

```bash
# Install dependencies
npm install
```

## Usage

```bash
# Run app
npm run dev

# Build app
npm run build

# Lint files
npx eslint "src/**/*.{ts,vue}"
```
