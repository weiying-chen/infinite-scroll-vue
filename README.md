# Vue Infinite Scrolling

Search population by city or state.

![](https://github.com/weiying-chen/vue-is/blob/main/screenshot.png)

**[Live Demo](https://vue-is.vercel.app/)**

## Installation

```bash
# Install dependencies
npm install
```

## Usage

```bash
# Run app
npm run dev

# Build app
npm run build

# Lint files
npx eslint "src/**/*.{ts,vue}"
```
